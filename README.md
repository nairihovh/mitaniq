# mitaniq


TO INSTALL ALL PREREQUISITES

```
npm install
```

START PROJECT

```
npm start
```

it is starting server on localhost on 3000 port

```
NEED TO CREATE keys FOLDER AND ADD PRIVATE_KEY AND CERTIFICATE
```
